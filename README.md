# BeRevisable

Make active records revisable

## Installation

Add this line to your application's Gemfile:

    gem 'be_revisable',     
    :git => "http://github.com/Natural-Intelligence/be_revisable.git",
    :branch => 'v0.0.1'

And then execute:

    $ bundle
